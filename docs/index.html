BiteLogは、食事記録と栄養管理を簡単に行えるiOSアプリです。
日々の食事を記録し、栄養バランスをチェックしましょう。

今すぐダウンロード 機能を見る

主な機能

📝

簡単食事記録

食品を選んで分量を入力するだけで、簡単に食事を記録できます。

📊

栄養成分表示

カロリー、タンパク質、脂質、糖質、食物繊維の情報を自動計算。

📅

日別管理

朝食、昼食、夕食、間食に分けて食事を管理できます。

🥗

食品マスター

よく使う食品を登録して、効率的に食事記録を行えます。

📈

データエクスポート

記録したデータをCSV形式でエクスポートして分析できます。

🌏

多言語対応

日本語と英語に対応し、お好みの言語で利用できます。
